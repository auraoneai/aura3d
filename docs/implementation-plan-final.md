## Iteration Loop
For every iteration:

1. Pick incomplete rows from `docs/requirements-trace.md`.
2. Assign rows to workstreams.
3. Implement missing production behavior.
4. Add or strengthen tests.
5. Run focused gates.
6. Update trace rows with evidence.
7. Run full verification.
8. Repeat until `pnpm verify:trace` passes.
