## Visual Problems To Fix

- Scenes look procedural and simple.
- Many objects read as generated test geometry, not premium product assets.
- Material response does not yet read as high-end physical rendering.
- HDR/IBL/PMREM quality is not proven by current screenshots.
- Product shots do not yet look like commercial catalog renders.
- Material shots do not yet prove chrome, brushed metal, glass, clearcoat, fabric, ceramic, emissive, and rough plastic differences.
- Interior/architecture visual depth is not proven.
- Character/animation visual quality is not proven.
- Postprocess is not yet proven as a production visual pipeline.
- Three.js comparisons prove setup structure more than final visual quality.

## V4 Rule

No V4 flagship scene can be accepted if it looks like the V3 screenshots with only a new filename.
