# V4 Product Configurator

Start from `@galileo3d/engine`, call `createG3DApp`, then use `workflows.productConfigurator` with a GLB product, material variants, contact shadows, diagnostics, and screenshot capture.

Evidence: `examples/external-product-configurator`, `apps/product-studio-pro`, and `tests/reports/external-gallery/product/external-product-configurator.png`.

Boundary: the workflow is product-visualization focused and does not claim full Three.js API replacement.
