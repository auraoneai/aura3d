# Browser And Hardware Matrix

Version: 1.0.10

## Aura3D advantage

Browser evidence is generated by Playwright tests and release tools. The current repository should not claim a broad browser/device matrix unless the relevant generated report exists and passes in the environment being cited.

## Current Sources

- `playwright.config.ts`
- `tests/browser/`
- `tests/reports/browser.json`
- `tests/reports/final-browser.json`
- `tests/reports/webgpu-hardware-matrix.json`
- `docs/rendering/webgpu-hardware-matrix.md`

## WebGL2

WebGL2 is the primary backend used by most browser routes. Local Playwright success proves the configured browser/project only.

## WebGPU

WebGPU support depends on `navigator.gpu`, adapter/device availability, browser flags, OS, and hardware. The WebGPU hardware matrix report should be regenerated before making a WebGPU support claim.
