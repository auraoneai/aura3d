# Competitive Positioning

Aura3D does not replace every browser 3D tool. It gives AI coding agents a
smaller, stable, deployment-aware surface to write against.

Use Aura3D when the job is agent-written browser 3D with typed assets,
templates, diagnostics, screenshots, and static deployment checks.

Use manual renderer code, framework-specific renderer layers, Threlte, TresJS, Theatre.js, or Troika when
the team specifically needs their lower-level control, framework-native
ecosystem, animation tooling, or focused helper surface.
