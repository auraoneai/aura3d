## Scoped Release Copy

Allowed:

- Aura3D helps AI coding agents build browser 3D scenes with public TypeScript APIs.
- Aura3D uses typed GLB/glTF assets generated by the CLI.
- Aura3D includes prompt-plan guidance, scene kits, diagnostics, screenshots, and deployment checks.
- Round 50 has a scoped SDK/product-context local/private-beta artifact.

Not allowed from this track alone:

- Aura3D beats manual renderer code.
- Aura3D passed the frozen external AI-agent benchmark.
- Aura3D has neutral benchmark superiority proof.
- Any public npm release has occurred.

## Frozen Benchmark-Superiority Track

The frozen benchmark-superiority track is governed by
`docs/project/frozen-benchmark-release-gates.md` and `benchmark/protocol.md`.
It remains blocked for Round 50 because external neutral review and external
score files were owner-skipped for scoped move-on only.

The scoped owner bypass does not narrow or satisfy the frozen benchmark bar. It
only permits the scoped product-context workstream to move forward without
claiming benchmark superiority.
