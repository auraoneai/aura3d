Keywords: install scaffold build deploy debug migrate.

## Install

`pnpm add @galileo3d/engine`

## Scaffold

`pnpm create g3d`

## Build

`pnpm build`

## Deploy

Serve the static output generated by the selected template.

## Debug

Use readiness reports under `tests/reports/`.

## Migrate

Run `pnpm three-compat:migration`.

## Snippet 1

```ts
import { createRendererV5 } from "@galileo3d/rendering";
import { SceneCompat } from "@galileo3d/three-compat";

const renderer = createRendererV5({ width: 800, height: 600 });
const scene = new SceneCompat();
console.log(renderer.captureScreenshot(), scene.type);
```
