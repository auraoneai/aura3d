# Support Policy

Version: 1.0.0

## Supported Surface

Support applies to current package exports, app routes, templates, tests, and docs that still exist in this repository.

## Current Support Areas

- Package import and build issues.
- Renderer, asset, animation, physics, controls, input, workflow, and editor-runtime APIs.
- Local app routes under `apps/`.
- Templates under `templates/` and `packages/create-aura3d/templates/`.
- Generated report commands documented in `docs/project/verification-evidence.md`.

## Unsupported Or Evidence-Limited Areas

- Full Unity/Unreal replacement claims.
- Full low-level renderer code API drop-in compatibility.
- Full glTF ecosystem coverage.
- Full WebGPU device/browser matrix coverage.

## Issue Triage

Support wording and public-release notes are governed by `docs/project/product-studio-claim-registry.md`.
