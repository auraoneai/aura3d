# Apps Classification

The root `/index.html` route is the public starter example registry. It lists
only the routes that teach the compact agent API directly.

## Starter Example Registry

| Route | Status | Use |
| --- | --- | --- |
| `/apps/hello-world-typed-asset/` | starter example | Typed asset refs with `model(assets.robot)` and `lights.studio()`. |
| `/apps/material-lighting/` | starter example | Material helpers and light primitives. |
| `/apps/camera-path/` | starter example | Camera path, timeline, lighting, and effects helpers. |

## Policy

The root registry lists only the three starter example routes. Marketing may
embed retained engine evidence when the page labels it as engine evidence and
keeps the starter path separate from polished proof routes.
