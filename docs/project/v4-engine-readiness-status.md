Status: in progress.

The current product position is blocked from broad parity or replacement claims. G3D is being reset to a TypeScript/WebGL rendering SDK for controlled product, material, asset-viewer, and clean scene-rendering workflows.

## Current Gate

The only gate that can unblock public demos is:

```sh
pnpm engine-readiness:root
```

Public examples remain disabled until this command passes with fresh canonical screenshot evidence.

- Unreal replacement
- broad Three.js replacement

- production game engine
- full WebGPU engine
- full glTF implementation
- full postprocess suite
