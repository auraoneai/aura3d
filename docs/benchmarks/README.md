# Benchmark Reports

This directory stores human-readable benchmark and comparison reports.

Current Agent 6 reports are scaffold reports generated by:

```sh
pnpm exec tsx --tsconfig tsconfig.base.json tools/compare-engines/index.ts --write-reports
```

The scaffold verifies scene equivalence across Galileo3D, Three.js, and Babylon.js definitions. The generated reports also pin the local Node, package manager, OS, hardware, Playwright Chromium audit browser, browser settings, dependency versions, browser WebGL2 microbenchmark samples, measured benchmark bundle artifacts, failure-log fields, audit screenshots, same-corpus glTF compatibility linkage, and feature-comparison coverage.

It still does not measure rendered production-scene parity, rendered scene screenshots, GPU counters, public package release bundles, or visual output parity for external Three.js/Babylon.js glTF loader output, so it does not support stronger public claims.

`docs/benchmarks/pbr-rendering-comparison.md` is separate from the scaffold reports. It records one bounded perspective-camera WebGL2 PBR comparison scene against a same-page Three.js reference, including Galileo/reference/diff PNG artifacts and delta metrics. That report does not change the broader limits above.
