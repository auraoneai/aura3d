# Aura3D And React Three Fiber

Use Aura3D when framework-neutral source code, small agent-readable examples,
typed asset deployment, and static hosting checks matter most.

Use React Three Fiber when the app is already deeply React-native and the team
wants the R3F/Drei ecosystem, reconciler semantics, and direct Three.js object
access.

Aura3D includes a thin React adapter, but it is not an R3F replacement.
