# Aura3D And Raw Three.js

Use Aura3D when an AI coding agent needs a smaller, stable, documented scene
API plus typed asset deployment and verification.

Use raw Three.js when the team needs full engine-level control, custom render
internals, the broadest plugin ecosystem, or direct access to every Three.js
primitive.

Aura3D's value is reliability for agent-written app code, not rendering things
that Three.js cannot render.
