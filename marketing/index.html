01 — Agent API

Small imports, real code.

The public surface is not a hidden runtime generator. Coding agents write TypeScript against a compact scene API, then Aura3D handles the canvas lifecycle, asset refs, diagnostics, screenshots, and deploy checks.

agent-scene.ts typed-assets.ts route-health.ts
TypeScript 
// The agent writes source code. Aura3D keeps the surface small.
import {
  createAuraApp, scene, model, camera,
  lights, effects
} from "@aura3d/engine";
import { assets } from "./aura-assets";

createAuraApp("#app", {
  diagnostics: { overlay: true },
  scene: scene({ background: "#05070a" })
    .add(model(assets.robot, { scale: 1.2 }))
    .add(lights.studio({ intensity: 1.2 }))
    .camera(camera.orbit({ distance: 4 }))
    .add(effects.bloom({ intensity: 0.35 }))
});
// model("robot") is rejected by the safe API.

06 — Runtime

Honest about what runs.

Aura3D should fail visibly when a canvas, backend, or asset is missing. WebGL2 remains the broad browser path, while WebGPU is conditional evidence, not a universal claim.

● Default browser path

WebGL2

The practical default for agent-written browser 3D: canvas setup, resize, render loop, camera, lights, diagnostics, screenshots, and cleanup handled by the app API.

● Conditional evidence

WebGPU

WebGPU work remains evidence-bound. If the browser exposes navigator.gpu, routes can prove support. If not, diagnostics must name the unsupported state.

07 — Deployment

Bring your assets.
Do not invent paths.

The CLI fingerprints user files, writes the manifest, generates typed asset refs, and checks static output before upload. Agents should read those generated refs instead of guessing URLs.

Add a real asset
# The file already exists on disk.
npx @aura3d/cli@latest assets add \
  ./assets/robot.glb \
  --name robot \
  --public-path /aura-assets/

# Generated files:
./aura.assets.json
./src/aura-assets.ts
Reference typed refs
import { assets } from "./aura-assets";

scene()
  .add(model(assets.robot))
  .add(lights.studio());

// Safe path: model(assets.robot)
// Rejected path: model("robot")
