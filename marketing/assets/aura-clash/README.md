# Aura Clash marketing assets

This folder mirrors the registered Aura Clash public GLB assets and generated typed asset module for marketing-site previews and launch screenshots.

The runtime source of truth remains `apps/aura-clash-showcase/public/aura-assets/` and `apps/aura-clash-showcase/src/aura-assets.ts`.

These assets are copied here so the marketing site can build poster cards, lazy previews, and case-study imagery without depending on local development paths.
