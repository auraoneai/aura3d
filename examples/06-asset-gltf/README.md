# 06 Asset glTF

## Purpose
Exercise public asset manager, registry, cache, and `GLTFLoader` APIs in the glTF roadmap slot.

## Run Command
Run a TypeScript-aware dev server from the repository root and open `examples/06-asset-gltf/index.html`.

## Acceptance Target
A loaded inline glTF triangle descriptor is visible and cache-backed public glTF loading succeeds.
