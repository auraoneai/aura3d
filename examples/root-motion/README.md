# Root Motion Example

This runnable example extracts root-motion deltas from `root-motion-run`, applies the same sample to a scene-style target and an ECS-style target, and keeps local stride animation sampled separately through `AnimationMixer`.

Run it from the repository dev server at `examples/root-motion/index.html`. Browser state is exposed as `window.__AURA3D_ROOT_MOTION_EXAMPLE__` for smoke tests and manual validation.
