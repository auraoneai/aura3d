# 02 Materials PBR

## Purpose
Exercise the public `PBRMaterial` API in the PBR roadmap slot.

## Run Command
Run a TypeScript-aware dev server from the repository root and open `examples/02-materials-pbr/index.html`.

## Acceptance Target
A material comparison grid is visible and each cell is backed by a public `PBRMaterial` roughness/metalness configuration.
