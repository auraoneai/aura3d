# 07 Input Controls

## Purpose
Prove public input snapshots, orbit controls, and first-person controls can update camera-like public transform objects.

## Run Command
Run a TypeScript-aware dev server from the repository root and open `examples/07-input-controls/index.html`.

## Acceptance Target
Pointer drag and wheel input update orbit camera state, first-person controls move a camera, and current control metrics are visible.
