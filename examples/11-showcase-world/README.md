# 11 Showcase World

## Purpose
Show the broadest current Aura3D capability in one public example instead of isolated roadmap slices.

## Run Command
Run a TypeScript-aware dev server from the repository root and open `examples/11-showcase-world/index.html`.

## Acceptance Target
A rich interactive-style scene is visible with public rendering, scene, physics, animation, glTF asset, input, audio, editor runtime, and particle metrics exposed on `window.__AURA3D_EXAMPLE__`.
