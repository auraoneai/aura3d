# Aura3D Portfolio

This page is the user-facing examples index for the current engine capability slice.

Open:

```text
examples/index.html
```

The portfolio intentionally surfaces the strongest current flagship examples and hides the old numbered validation slices from the main examples list. The numbered examples remain in the repository as verification fixtures until their PRD and trace evidence is migrated.

It also includes a `Local Renderer Proofs` section for the implementation work that is locally ready but not fully Unity/Unreal-certified:

- product visual local proof
- PBR local proof
- HDR render-target local proof
- shadow-map local proof
- postprocess-suite local proof
- glTF parity guardrail
- WebGPU claim-boundary guardrail

The portfolio does not claim production readiness, broad Three.js superiority, or Unity/Unreal replacement status.
