# Held-Back Create-Aura3D Templates

These directories used to live under `packages/create-aura3d/templates/`, but they are not active starter templates for the current public scaffold command.

Active scaffold templates are limited to:

- `product-viewer`
- `cinematic-scene`
- `mini-game`

The archived directories here are preserved as historical or future-reference work. They are not copied by `create-aura3d`, not included in `@aura3d/create-aura3d` package files, and not part of the public starter-template surface.

Do not move one of these directories back into `packages/create-aura3d/templates/` unless the corresponding template is promoted with docs, tests, scaffold smoke coverage, and package manifest changes in the same patch.
