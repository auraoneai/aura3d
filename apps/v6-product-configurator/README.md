# G3D V6 Product Configurator

Real V6 app for product visualization. It loads imported GLB assets from the V6 corpus, binds a real HDR environment, renders through the public `@galileo3d/engine/v6` SDK (`loadGltfScene`, `loadHdrEnvironment`, `createProductViewer`), exposes `window.__g3dV6Runtime`, and supports a configuration interaction without using Three.js as the product runtime.
