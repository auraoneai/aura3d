# Claims And Boundaries

Allowed claims:

- Aura3D is a library and asset deployment pipeline for agent-written browser 3D.
- Agents write normal TypeScript/JavaScript source code.
- Users bring assets; Aura3D validates, hashes, indexes, type-generates, and
  checks deployment.
- Starter templates cover product viewer, cinematic scene, and mini game.
- Devtools expose asset state, FPS, draw calls, render size, backend, warnings,
  and errors.

Do not claim Aura3D is an AI model, an asset marketplace, or a substitute for
full engine-level control when manual renderer code or a framework-specific layer is a
better fit.
