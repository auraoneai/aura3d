# Prompt 04: Neon Tunnel Flythrough

Build a neon tunnel flythrough: procedurally generated tube geometry with emissive segments, camera animated through the tunnel, bloom postprocess, fog falloff.

## Assets

None. This prompt must be built procedurally.

## Required Visual Evidence

- A tunnel or tube interior must be visible.
- Emissive segments must be visible along the tunnel.
- Bloom or glow must be visible.
- Fog or depth falloff must be visible.
- Camera animation must be captured in runtime evidence, and the screenshot must read as a flythrough scene.
