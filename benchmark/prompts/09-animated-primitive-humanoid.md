# Prompt 09: Animated Humanoid Walk

Build an animated humanoid character walking across the ground plane. The result must read as one connected character, not as an exploded primitive puppet or rough prototype output.

## Assets

No prompt-provided or downloaded assets are allowed. Library-bundled humanoid helpers or built-in character assets exposed through the public context are allowed because they are part of the library being benchmarked. manual renderer code output must not fetch or invent external character assets.

## Required Visual Evidence

- The figure must read as one connected humanoid.
- Head, torso, arms, legs, hands, and feet must be visible.
- A ground plane must be visible.
- Feet must appear planted or grounded, with a grounding shadow or contact cue.
- Walk-cycle movement must be visible in runtime evidence.
- The screenshot must show a pose that implies animation, not a static stick figure.
- Limbs must not float away from the torso.
- The scene must not look like an exploded primitive assembly.
