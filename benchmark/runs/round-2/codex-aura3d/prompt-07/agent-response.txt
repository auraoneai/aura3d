Implemented the Material Lab in `./source`.

It uses procedural Aura3D primitives only: five visible spheres for metal, glass, rubber, emissive, and clearcoat materials, plus studio softbox cards, reflective floor/backdrop cues, bloom, studio/ambient/directional/point lights, and orbit controls.

Verification run:
- `npm run typecheck`
- `npm run build`

Build command:
```bash
cd source && npm run build
```

Run command:
```bash
cd source && npm run dev
```

Assumptions:
- No external assets are needed or used.
- Environment lighting/reflection is represented through Aura3D studio lighting, reflective materials, visible softbox/reflection cards, and the studio backdrop/floor.
