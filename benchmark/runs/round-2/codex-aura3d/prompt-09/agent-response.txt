Implemented the procedural 3D app in `source/src/main.ts`.

It renders a primitive humanoid with a sphere head, cylinder torso, box arms/legs/feet, visible ground plane, grid, footprints, shadows, and a continuous walk cycle moving across the plane with swinging limbs and body bounce.

Build command:
```bash
cd source && npm run build
```

Run command:
```bash
cd source && npm run dev
```

Verification run:
```bash
cd source && npm run typecheck
cd source && npm run build
```

Assumptions: no external assets are used; the character is fully procedural. I did not leave a dev server running.
