Implemented the procedural city block app in `source`.

Build command:
```bash
npm run build
```

Run command:
```bash
npm run dev
```

Verification completed:
```bash
npm run build
node node_modules/@aura3d/engine/dist/aura3d-cli/cli.js check-deploy --dist dist
```

Assumptions:
- No external assets are used; the scene is fully procedural.
- The empty `aura.assets.json` and `src/aura-assets.ts` are included only so Aura3D deployment checks pass for a no-asset prompt.
- I did not leave a dev server running.
