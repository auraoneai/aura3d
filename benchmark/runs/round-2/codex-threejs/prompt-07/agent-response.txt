Implemented the Material Lab in `source/src/main.ts`.

It includes five procedural spheres for metal, glass, rubber, emissive, and clearcoat materials, plus visible studio softboxes, shadows, procedural environment reflections, and enabled `OrbitControls`.

Verified with:

```bash
npm run build
```

Build command:

```bash
npm run build
```

Run command:

```bash
npm run dev
```

Assumptions: `llms.txt` was not present in the provided directory or context bundle, so I used the local Three.js context and prompt requirements only. No assets were used.
