# Prompt 05: 3D Data Visualization

Build a 3D data visualization: a 6x6 grid of bars whose heights animate from random values, color by height, hover-highlight, orbit camera, axis labels.

## Assets

None. This prompt must be built procedurally.

## Required Visual Evidence

- Exactly or clearly approximately 36 bars must be visible.
- Bar heights must vary.
- Color must correspond to height.
- Axis labels must be visible and readable.
- Hover-highlight behavior must be implemented and documented in the run notes.
