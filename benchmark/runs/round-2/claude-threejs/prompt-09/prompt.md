# Prompt 09: Animated Primitive Humanoid

Build an animated character placeholder: a humanoid figure built from primitive shapes (sphere head, cylinder body, box limbs) with a procedural walk-cycle animation moving across the ground plane.

## Assets

None. This prompt must be built procedurally.

## Required Visual Evidence

- The figure must read as humanoid.
- Head, torso, arms, and legs must be made from primitives.
- A ground plane must be visible.
- Walk-cycle movement must be visible in runtime evidence.
- The screenshot must show a pose that implies animation, not a static stick figure.
