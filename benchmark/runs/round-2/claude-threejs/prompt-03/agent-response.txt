Done. The procedural solar system is built, type-checks, and builds cleanly.

## What was built

A Three.js + TypeScript + Vite app in `source/` (`src/main.ts`, plus label styles in `index.html`):

- **Sun** — bright unlit sphere at the origin, an additive radial-gradient halo sprite, and a `PointLight` that lights the planets.
- **6 planets** — Mercura, Venora, Terran, Marnis, Jovian, Saturnis — each at a clearly different orbital distance (12 → 18 → 25 → 33 → 46 → 60) and a different orbital speed (faster when closer). Each spins on a tilted axis; Saturnis has a ring. Faint orbit-ring lines make the distinct distances read even when a planet is behind the sun.
- **Labels** — readable HTML labels via `CSS2DRenderer`/`CSS2DObject` on a transparent overlay, so they stay crisp and are never washed out by bloom. Each planet and the sun is labeled.
- **Bloom** — `EffectComposer` + `RenderPass` + `UnrealBloomPass` with a 0.35 threshold, so only the bright sun glows while planets stay sharp.
- **Orbit camera** — `OrbitControls` with damping, positioned at (0, 42, 90) to frame the whole system out to the furthest orbit; resize-aware.

Built procedurally — no external assets. `@types/three@0.165.0` was added to `devDependencies` so the strict `tsc` typecheck passes.

## Commands

- **Install:** `npm install`
- **Build:** `npm run build` (runs `tsc --noEmit` then `vite build` — passes)
- **Run:** `npm run dev -- --port <assigned-port>`
