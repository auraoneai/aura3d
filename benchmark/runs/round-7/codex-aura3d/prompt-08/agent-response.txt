Implemented in `./source`.

Build completed successfully with:

```bash
npm run build
```

Run command:

```bash
npm run dev
```

Assumptions:
- No external assets are used.
- City geometry comes from the public Aura3D `prefabs.cityBlock({ blocks: 20, litWindows: true })`.
- The day/night toggle rebuilds the scene with different sky color, fog, bloom, and lighting.
