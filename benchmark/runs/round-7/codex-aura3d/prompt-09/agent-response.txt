Implemented the procedural animated humanoid app in `source/src/main.ts` using Aura3D’s `prefabs.primitiveHumanoid()` with studio lighting, perspective camera, visible ground/path cues from the prefab, and a looped 4-second timeline for the walk cycle.

Build completed successfully.

Build command:
```bash
npm run build
```

Run command:
```bash
npm run dev
```

Assumptions: no external assets are needed; the Aura3D humanoid prefab satisfies the primitive head, torso, arms, legs, ground plane, and procedural walk-cycle requirements.
