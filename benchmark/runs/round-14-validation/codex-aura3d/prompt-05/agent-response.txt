
```bash
cd source && npm run dev
```

Assumptions:
- Aura3D `prefabs.dataBars3D({ grid: 6 })` provides the 36 renderable bars, varied animated heights, color-by-height chart geometry, axis/tick/legend scene elements, and chart hover metadata.
- Hover-highlight is implemented with `interactions.raycastHover(...)` and documented in the on-screen readout text.
- No external assets are used; the scene is procedural only.
