## Done

Built a procedurally generated city block in `source/src/main.ts` using raw Three.js (v0.165.0) + `OrbitControls`. The build completed successfully (`tsc --noEmit` clean, Vite bundle produced).

### What it renders
- **20 buildings** — a deterministic 4×5 grid of box buildings (seeded PRNG, so identical every run).
- **Varying heights** — each building is 3–14 floors (≈9–42 units tall), randomized.
- **Windows** — per-building procedural canvas textures: a color map (concrete wall + dark glass panes/frames) on the 4 side faces, plus a matching emissive map where ~55% of windows are "lit" so they glow at night.
- **Streets** — dark asphalt ground reads as the road grid between raised sidewalk lots, with dashed yellow lane center-lines along every street corridor.
- **Street lights** — poles with arm + emissive lamp heads and `PointLight`s placed along the street corridors (off by day, on at night).
- **Day/night toggle** — a fixed DOM button swaps a full lighting/sky state: gradient sky-dome colors, scene background, fog color, hemisphere/sun/ambient intensities (sun→moonlight), tone-mapping exposure, window emissive glow, street-light point lights + lamp emissive, and a star field that fades in at night.
