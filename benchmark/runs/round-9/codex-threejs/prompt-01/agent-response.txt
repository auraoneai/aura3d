Implemented the browser 3D physics playground in `source/src`.

Build completed successfully with:

```bash
cd source && npm run build
```

Runner command:

```bash
cd source && npm run dev
```

Assumptions: raw Three.js context takes precedence per `llms.txt`; no external assets were used. The physics is procedural and self-contained: 50 cubes, tilted ramp, visible contact markers, reset button, live contact count overlay, and OrbitControls camera.
