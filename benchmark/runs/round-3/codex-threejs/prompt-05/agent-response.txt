Implemented the 3D data visualization in `./source`.

Build command:
```sh
cd source
npm install
npm run build
```

Run command:
```sh
cd source
npm run dev -- --port 5177
```

Notes:
- Uses procedural Three.js geometry only, no assets.
- Renders a 6x6 grid of 36 animated bars with varying heights.
- Bar color maps to height from blue/green through orange.
- Orbit camera is enabled.
- Axis labels and tick labels are visible via CSS2D labels.
- Hover-highlight is implemented with raycasting: hovered bars turn white/emissive and show a floating value label plus HUD readout.
- I verified with `npm run build` and a headless Chrome screenshot; the dev server was stopped afterward.
