Implemented the particle fountain app in `source/src/main.ts`.

It uses procedural Aura3D primitives/effects only: visible emitter nozzle, collision ground plane, animated upward/falling fountain particles, lifetime color key, and a functional emission-rate slider that updates particle density.

Build command:
```bash
cd source && npm run build
```

Run command:
```bash
cd source && npm run dev
```

Verification:
```bash
npm run build
```
passed.

Assumptions:
- No external assets are needed or used.
- The Aura3D CLI deploy check could not run because `@aura3d/cli@latest` was not available from the configured npm registry.
