Implemented the procedural Material Lab in `source/src/main.ts`.

It contains five visible sphere swatches for metal, glass, rubber, emissive, and clearcoat-style materials, plus studio softbox panels, reflection bands, floor/backdrop, bloom, controlled lights, and Aura3D orbit interaction.

Build command:
```bash
npm run build
```

Run command:
```bash
npm run dev -- --port 4177
```

Current dev URL:
```text
http://127.0.0.1:4177/
```

Verification: `npm run build` passes.

Assumption: Aura3D’s public safe API exposes PBR/emissive material controls but not a direct HDR environment-map field, so the environment/reflection requirement is represented procedurally with visible emissive softboxes, reflection cards, and glossy studio reflection bands.
