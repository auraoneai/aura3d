Assumptions:
- No external assets were used.
- I used procedural Three.js primitives because the local Aura3D public primitive API exposes `box`, `sphere`, and `plane`, but not cylinder or per-frame primitive transform animation. The scene includes a sphere head, cylinder torso, box limbs, visible ground plane, ghosted prior poses, and a looped walk cycle moving across the ground.
