Implemented the mini-golf app in `./source`.

It includes a procedural Aura3D course with a green, rails, one obstacle, cup, ball, aim marker, pointer interaction, score HUD, and `camera.follow` targeting the ball. The playable overlay supports click-drag aim, shooting, friction, wall/obstacle collision, cup detection, and stroke counting.

Build command:
```bash
npm run build
```

Run command:
```bash
npm run dev
```

Current dev server:
```text
http://127.0.0.1:5175/
```

Assumptions: no external assets were used; the game is built procedurally as required.
