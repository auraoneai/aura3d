# Prompt 02: Particle Fountain

Build a particle fountain: gravity-affected particles emitted upward from a point, colored by lifetime, with a ground plane they collide against and a controllable emission rate.

## Assets

None. This prompt must be built procedurally.

## Required Visual Evidence

- The fountain emitter must be identifiable.
- Particles must form an upward-and-falling arc, not static random dots.
- Lifetime color variation must be visible.
- A ground plane must be visible.
- An emission-rate control must be visible.
