Implemented the Material Lab app in `source/src/main.ts`.

It creates a procedural Three.js studio scene with five visible spheres: metal, glass, rubber, emissive, and clearcoat. The scene includes PMREM environment lighting via `RoomEnvironment`, visible studio light panels, soft shadows, reflective wall markers, and enabled `OrbitControls`.

Build command:
```bash
cd source
npm run build
```

Run command:
```bash
cd source
npm run dev
```

Verification: `npm run build` passes. I also rendered it locally with Vite and checked a browser screenshot.

Assumptions: `llms.txt` was not present in the provided bundle, and no Aura3D package/API files were available in `source` or `context`, so I used the provided Three.js/Vite scaffold and built the prompt procedurally with no assets.
