Implemented the 3D data visualization in `./source`.

Build command:
```sh
cd source && npm run build
```

Run command:
```sh
cd source && npm run dev
```

Verification:
`npm run build` completed successfully.

Assumptions:
- `./context/llms.txt` and `./context/docs/agents/benchmark-recipes.md` were not present, so I used the provided three.js scaffold/context.
- No external assets were used.
- Hover-highlight is implemented with raycasting: hovering a bar brightens it, slightly enlarges its footprint, and shows an `X/Z/height` tooltip.
