# Prompt 03: Procedural Solar System

Build a procedural solar system: sun, 6 planets at different orbital distances and speeds, label each planet, bloom on the sun, orbit camera.

## Assets

None. This prompt must be built procedurally.

## Required Visual Evidence

- One sun and 6 planets must be visible.
- Orbit distances must be visibly different.
- Planet labels must be visible and readable.
- Sun glow or bloom must be visible.
- Orbit camera setup must frame the whole system.
