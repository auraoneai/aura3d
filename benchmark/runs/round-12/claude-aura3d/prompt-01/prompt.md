# Prompt 01: Physics Playground

Build a physics playground: 50 falling cubes onto a tilted ramp with collision response, camera orbit, reset button, and live contact count overlay.

## Assets

None. This prompt must be built procedurally.

## Required Visual Evidence

- Many cubes must be visible.
- A tilted ramp must be visible.
- Collision/contact behavior must be visible in either the screenshot state or captured route evidence.
- A reset control and live contact count overlay must be visible.
- Camera framing must make the physics setup understandable without reading source code.
