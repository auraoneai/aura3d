## Summary

Built a procedurally generated city block in raw Three.js (`source/src/main.ts`):

- **20 buildings** — a 5×4 grid of box meshes, each with deterministically varied width/depth and height (12–58 units), so heights clearly vary.
- **Windows** — a procedural canvas texture used as both `map` (glass tint by day) and `emissiveMap` (a deterministic lit pattern that glows at night); repeat counts scale per building so window rows track height.
- **Streets** — dark road planes running between every row and column with yellow center lines, over a green ground plane.
- **Street lights** — poles with arms and glowing lamp heads plus `PointLight`s that switch on at night.
- **Day/night toggle** — a fixed-position button swaps a full lighting/sky state set: gradient sky-dome colors, hemisphere/ambient/sun intensities + sun color, fog color, building window emissive glow, lamp emissive + point-light power, and a star field that appears only at night.

OrbitControls let the runner orbit the scene.
