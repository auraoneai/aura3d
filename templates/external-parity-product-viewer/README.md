# A3D ExternalParity Product Viewer Template

This template is the ExternalParity ExternalParity product viewer scaffold. It imports the public package API through `@aura3d/engine/apps`, creates a `createA3DApp` runtime, and renders the `scene-showcase` workflow with production quality settings.

```sh
npm install
npm run build
```

Claim boundary: this template proves installable scaffolding and external Vite build behavior. Same-scene Three.js visual parity remains a later ExternalParity milestone.
