# Support Policy

Version: 0.1.0-alpha.0

## Current Support Level

Galileo3D is an experimental TypeScript web 3D engine prototype. Support for version `0.1.0-alpha.0` is limited to repository issues, reproducible test cases, and documentation updates.

## Where To Ask For Help

- Use the bug report template for reproducible defects.
- Use the feature request template for scoped enhancement proposals.
- Include the package version, commit SHA, browser, OS, reproduction command, and relevant report JSON path.

## Response Expectations

This rebuild does not yet have production service-level commitments. Maintainers should prioritize:

- release verification failures;
- security reports;
- public claim or documentation contradictions;
- regressions in examples, tests, and package exports.

## Known Limits

Known limitations belong in `docs/known-limits.md` or the relevant v2 gate document. Public support responses must not claim production readiness or stronger competitive positioning unless `docs/v2/claim-registry.md` allows the exact wording.
